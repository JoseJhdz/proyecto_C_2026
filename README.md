# Proyecto C

## Descripción
Para este trabajo logramos crear un conversor de monde en el Lenguaje de C, donde tambien mostramos un proceso del como podemos ejecutarlo con diferentes comandos.

### Prerrequisitos

Para trabajar en este proyecto se necesito la instalacion de Build-essencial que es un paquete de compiladores que nos funciona para la compilacion de C/C++.
Tambien se necesita la maquina virtual que en este caso es VirtualBox, que contiene el sistema operativo Ubuntu basado en Linux. 
Tambien se instala el controlador de versiones Git para tener el control de cambios en archivos.

Para poder vizualizar todo el trabajo que se hace instalamos en nuestro ambiente de trabajo un editor de codigo que en este caso es el Virtual Studio Code.
